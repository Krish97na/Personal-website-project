Works

{Connecting social media to e-commerce: A cross site cold start product recommendation system using microblogging information

A recommendation system which recommends products to the users from e-commerce websites on social media platforms in cold-start situations. The users' social media features and and e-commerce features are turned into embeddings using recurrent neural networks and modified gradient boosting trees method, which are then leveraged by a feature based matrix factorization approach to recommend products from cold start.

Master Thesis: Robotic software development using DevOps

The aim of the thesis was to present the results of the usage of DevOps practices in robotic software development. 15 DevOps practices have been selected and mapped to the robotic software development process. The potential advantages and limitations of the proposed mapping’s implementation were presented.
